# Customer Churn Analysis

## Project Overview
This project analyses telecom customer data to identify factors that influence customer churn.

Using Python, Pandas, and Seaborn, the dataset is cleaned and explored to uncover patterns in customer behaviour.

## Dataset
Telco Customer Churn Dataset from Kaggle.

## Tools Used
Python  
Pandas  
Seaborn  
Matplotlib  
Google Colab  

## Key Insights
- Customers with month-to-month contracts have the highest churn rate.
- Higher monthly charges are associated with increased churn.
- Customers with longer tenure are less likely to leave.
